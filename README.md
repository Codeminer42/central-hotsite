# Central Hotsite - [Template](http://www.centralcm42.com/)


Boilerplate of the Central Hotsite build with Gulp

## Requirements:

- Nodejs (8.1.3)
- NPM (5.0.3)

## Usage:


```
 npm install && npm start
```
